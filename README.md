# Прототип: Визовый помощник Т-Путешествия

Интерактивный прототип по [spec.md](spec.md) в дизайне **Т-Банка** (тёмная тема, жёлтый акцент). Синхронизация через **Supabase**.

## Как открыть

Через локальный сервер (нужно для Supabase Auth и навигации):
```bash
cd prototype
python3 -m http.server 8080
```
Откройте в браузере: http://localhost:8080

## Supabase

1. Создайте проект на [supabase.com](https://supabase.com).
2. В **SQL Editor** выполните скрипт `supabase-schema.sql` (таблицы `profiles`, `documents`, `motivation_letters` и RLS).
3. В **Settings → API** скопируйте URL и anon key.
4. В `supabase-config.js` укажите:
   - `window.SUPABASE_URL = 'https://ВАШ_ПРОЕКТ.supabase.co'`
   - `window.SUPABASE_ANON_KEY = 'ваш_anon_key'`

Без настройки Supabase приложение работает с сохранением только в localStorage и кнопкой «Продолжить без входа».

## Экраны и сценарии

- **Авторизация** — при первом заходе: ввод email → отправка ссылки для входа (Supabase Auth, magic link). Либо «Продолжить без входа».
- **Новая заявка** — выбор страны (фото), тип визы, «Заполнить заявку» → форма и чек-лист. Кнопка **«Экспорт для подачи»** ведёт на отдельную страницу.
- **Мой профиль** — визовый профиль. При входе по почте сохраняется в Supabase (таблица `profiles`).
- **Документы** — загрузка документов (mock), список в localStorage.
- **Страница экспорта** (`export.html`) — открывается по «Экспорт для подачи»:
  - **Мотивационное письмо для США** — поля: цель поездки, связи с родиной, план поездки, текст письма. Сохранение в Supabase (`motivation_letters`).
  - **Чат с помощником** — вопросы по визам и команды (/help, /сша, /мотивационное и др.).
